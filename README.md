# QR Cube

A browser-based tool for generating 3D-printable cubes with unique QR codes engraved into each face.

**This is a prototype and may not work perfectly.** Expect rough edges.

## Features

- Assign 1–6 URLs across the 6 faces of a cube
- Real-time 3D preview with multiple view modes
- Adjustable cube size, engrave depth, and edge treatments (chamfer/fillet)
- STL export for 3D printing
- Light/dark theme with accent color presets

## Usage

Open `index.html` in a browser. No build step, no dependencies, no server required.
